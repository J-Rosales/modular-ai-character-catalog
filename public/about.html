About Botparts

A static, deterministic catalogue designed for Neocities hosting.

What this site does

Botparts is a static catalogue UI that reads pre-generated JSON manifests and renders lightweight character details without server-side processing.

What this site does not do

  • No data generation or schema mutation.
  • No build tooling or runtime API calls.
  • No dynamic filtering yet (planned for the next phase).
