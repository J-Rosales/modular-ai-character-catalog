License & Attribution

Every entry in this catalogue includes provenance details, redistribution status, and a link to the original source. Always review the character detail page before exporting or sharing.

Attribution expectations

  • Each profile retains its original author credit and source link.
  • Redistribution restrictions are honored in the download flows.
  • Backups are only hosted when redistribution is allowed.

Catalogue UI license

The catalogue interface is a static, client-side project. It contains no server-side logic and only consumes data exported by the generator pipeline.
