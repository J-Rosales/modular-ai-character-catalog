About the Catalogue

This project is a static, client-side catalogue of SillyTavern character bots. Every profile is backed by JSON data and includes attribution, licensing, and export-ready content.

What you can do here

  • Explore featured characters on the landing page.
  • Browse the full catalogue and filter by tags (coming soon).
  • Open any character to inspect modules, transforms, and download options.

License & Attribution

Every character includes an original source link. If redistribution is not allowed, we only link to the source and avoid hosting backups. Always review the provenance and licensing notes before sharing.

The catalogue UI is intended for static hosting and requires no server-side processing.
